A collection of Computer Vision projects and experiments using **Python** and **OpenCV**. This repository demonstrates various techniques for image processing, object detection, and motion analysis.

## üìÇ Projects Overview

### 1. Motion Analysis & Vehicle Tracking
* **Description:** Tracks moving vehicles in video feeds.
* **Techniques:** Background subtraction, contour detection, and centroid tracking.
* **Key Files:** `track_black_cars.py`, `count_white_cars.py`.

### 2. Object Detection (Haar Cascades)
* **Description:** Real-time object detection using pre-trained XML classifiers.
* **Techniques:** Haar Cascade Classifiers for car detection.
* **Key Files:** `haarcascade_car.xml`.

### 3. Stereo Vision & Disparity
* **Description:** Computing depth maps from stereo images (left/right pairs).
* **Techniques:** StereoBM (Block Matching) to calculate disparity maps.
* **Key Files:** `check_disparity.py`, `assignment 2/`.

### 4. Hough Transform
* **Description:** Detecting geometric shapes (lines/circles) in images.
* **Key Files:** `hough_transform.py`.
